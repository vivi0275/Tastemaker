# Protocole de test utilisateur — ICP DJs & Producteurs

Objectif : valider les pain points P1–P6 avec 5 testeurs (DJs ou producteurs), 3 artistes de référence chacun.

## Scénario (15–20 min)

1. Ouvrir Tastemaker (prod ou local).
2. Chercher **Artiste 1** (référence personnelle du testeur).
3. Parcourir le dig crate — tester filtres Likes / Reposts.
4. Lancer au moins une preview.
5. Cliquer **Dig deeper** → **Dig their crate** sur un artiste similaire.
6. Répéter pour **Artistes 2 et 3**.

## Questions ouvertes

1. Où as-tu abandonné ou été frustré ?
2. Repost ou like — lequel tu chasses en premier ?
3. Qu’est-ce qui te manque le plus pour remplacer ton flow actuel ?
4. Sur une échelle 1–5, à quel point serais-tu déçu si Tastemaker disparaissait ?
